INSTALLATION INSTRUCTIONS FOR UNITEX 2.1beta
-------------------------------------------------------------


WINDOWS
-------
You have nothing to do but running the Java graphical interface by double-cliking on
the file "Unitex.jar" located in the "App" directory.



LINUX/MacOS
-----------
You must compile the C++ sources. To do that, go into the "Src/C++/build" directory and launch
the command "make install". Then, you can launch the graphical interface with the following
command:

java -jar UNITEXPATH/App/Unitex.jar &
