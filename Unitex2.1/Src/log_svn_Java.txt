r792 | paumier | 2009-05-05 11:48:51 +0200 (Tue, 05 May 2009) | 1 line

Format .tfst has been modified in order to handle the notion of logical letter that is useful for Korean. Now, start and end offsets are represented by x.y.z where x is the position in tokens, y the position in characters in the current token and z is the position in logical letters in the current character. For Korean, it is useful to know that we are at n logical letters inside a syllabic character. For any other language, this last value should always be 0.
